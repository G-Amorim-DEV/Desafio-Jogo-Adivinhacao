# Desafio-Jogo-Adivinhacao
Jogo de Adivinhação focado em POO e SOLID. Estrutura dividida em classes: Jogador (dados), GerenciadorRanking (persistência JSON) e MotorJogo (lógica). Aplica SRP para separar responsabilidades e DIP com interfaces de dificuldade. Inclui validação de entradas, cálculo de score dinâmico e exibição de ranking ordenado.
